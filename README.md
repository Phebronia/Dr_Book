# 📚 Dr. Book

A modern online bookstore designed to make browsing and ordering books simple and enjoyable.

The website automatically loads books from Google Sheets, allowing the store owner to update the catalog without modifying the website code.

---

## ✨ Features

- 📚 Browse available books
- 🔍 Instant search by book title
- 🗂️ Filter books by category
- 📄 Automatically load books from Google Sheets
- 💬 Order directly through WhatsApp
- 📱 Responsive design for desktop and mobile
- 🎨 Clean and modern interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Google Sheets API (OpenSheet)
- GitHub Pages
